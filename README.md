# phase-0-gps-1

* ls - lists of files in a directory
* git checkout -b - creating and moving to new branch from the master branch in the repo
* git status - shows status of any commits needing to be made from edits
* commit - saving any changes
* add - setting up to make saves to any changes made
* touch - adding files to directory
* push origin - pushing the changes made to github on the repo
